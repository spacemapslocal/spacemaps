Space Maps

Sistema de Localização Indoor para Pessoas Cegas ou com Baixa Visão

Sobre o Projeto

A criação de um sistema de localização indoor para pessoas cegas e com baixa visão é essencial para garantir autonomia, segurança e inclusão em ambientes internos.

O sistema consiste em óculos inteligentes equipados com uma câmera conectada a um módulo ESP32, integrado a uma Inteligência Artificial especializada em reconhecimento de objetos e navegação espacial. A IA analisa o ambiente em tempo real e fornece instruções claras ao usuário por meio de comandos de áudio.

Com este dispositivo, pessoas cegas ou com baixa visão ganham independência para explorar espaços internos, reforçando o compromisso com a acessibilidade e a inclusão social.

Principais Características:

  • Óculos com câmera: captura imagens do ambiente para análise em tempo real.
  • Processamento com ESP32: garante eficiência e conectividade com a IA.
  • Inteligência Artificial: reconhece objetos e mapeia o espaço.
  • Feedback via áudio: fornece instruções claras e acessíveis ao usuário.
  • Foco na acessibilidade: projetado para atender às necessidades específicas.

Product Backlog - Space Maps

  1. Como pessoa com deficiência visual, quero receber instruções de áudio em tempo real para me locomover com segurança.
  2. Como usuário, quero que o sistema reconheça obstáculos para evitar colisões.
  3. Como pessoa com baixa visão, quero visualizar contrastes e ícones ampliados.
  4. Como usuário, quero conectar os óculos ao celular via Bluetooth para personalização.
  5. Como administrador, quero registrar diferentes mapas de ambientes internos.
  6. Como pessoa com deficiência visual, quero que o sistema funcione offline.

Sprint Backlog - Space Maps

  • Configurar módulo ESP32 e realizar testes de conectividade com a câmera.
  • Implementar algoritmo de detecção de obstáculos usando IA.
  • Desenvolver sistema de saída de áudio para fornecer instruções ao usuário.
  • Criar interface com alto contraste para usuários com baixa visão.
  • Programar integração do ESP32 com celular via Bluetooth.
  • Implementar sistema de armazenamento offline para funcionamento sem internet.
  • Criar protótipo inicial e realizar testes em ambiente controlado.

Sprint Review

Durante a Sprint Review, a equipe apresentou o protótipo dos óculos inteligentes com comandos de áudio básicos. Foi demonstrado o reconhecimento de obstáculos simples em ambiente controlado. O feedback recebido reforçou a necessidade de ampliar a base de objetos reconhecíveis e otimizar o tempo de resposta da IA.

Sprint Retrospective

A equipe concluiu que a comunicação interna funcionou bem e os papéis estavam claros. Como melhoria, ficou definido que será necessário dedicar mais tempo para testes integrados entre hardware e software. Também foi sugerido o uso de ferramentas de gestão ágil para melhorar a visibilidade do progresso.
